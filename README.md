# IonRod

A new craftable reactor rod for Subnautica.  
The **IonRod** is a more powerful variant of the standard reactor rod, offering dramatically increased energy output for late‑game bases and nuclear setups.

---

## Features

- **Increased power output** compared to standard reactor rods  
- **Longer lifespan** for extended base operation   
- Balanced for mid‑to‑late game progression  
- Works in all vanilla nuclear reactors

---

## Installation

1. Install [BepInEx](https://www.openmods.net/mod/31?gameId=1)
2. Install [Nautilus](https://www.openmods.net/mod/30?gameId=1)
3. Download the latest release  
4. Extract to your Subnautica *BepInEx/plugins* folder  

---

## Crafting

- Found in: **Fabricator → Resources → Electronics**  
- Recipe:  
  - 2 × Ion Crystal  
  - 1 × Glass
  - 2 × Titanium

---

## Compatibility

- Subnautica 2.0+  
- BepInEx 5.x  
- Nautilus 1.x  
- Works alongside other nuclear‑related mods
